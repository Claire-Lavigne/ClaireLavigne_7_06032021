**Les Petits Plats** is a responsive **recipe search** website built with a focus on **algorithmic optimization** and **search performance**.

<img src="https://github.com/Claire-Lavigne/repo-images/blob/main/LesPetitsPlats.JPG" width="80%">

## Getting Started
- `npm i`
- `npm run build`
- `npm run dev`

Switch to branch "algorithm2" to check the second algorithm

## Features implemented
✔️ Develop an **efficient search algorithm** for recipe filtering  
✔️ Search by ingredients, appliances, or utensils  
✔️ Build a **responsive and accessible interface**
